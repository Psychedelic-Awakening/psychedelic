# Psychedelic Institute App

Astro app for rebuilding the recovered Psychedelic Institute site and funnel pages.

## Routes

- `/`: rebuilt home page shell
- `/membership`: recovered membership tiers page
- `/guardian`: recovered guardian intake route
- `/ty-g`: recovered thank-you route

## Commands

- `pnpm dev`: start the Astro dev server
- `pnpm build`: build the site
- `pnpm preview`: preview the production build

## Environment

- `PUBLIC_GUARDIAN_FORM_ENDPOINT`: optional external endpoint for guardian application submissions
